## 📌 What This Project Does

Smart Contact Manager is a full-stack web application that allows users to securely manage their contacts. It provides features like user authentication, adding, updating, deleting, and searching contacts through a clean and responsive interface, backed by RESTful APIs.

- Hibernate / JPA
- MySQL
