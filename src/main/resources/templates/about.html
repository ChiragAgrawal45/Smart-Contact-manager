About Smart Contact Manager

Smart Contact Manager (SCM 2.0) is a powerful web application designed to help users efficiently manage their personal and professional contacts.

Built using Spring Boot, Thymeleaf, and MySQL, this application provides a secure and scalable environment for storing important contact information.

Users can easily add, update, delete, and organize contacts along with profile pictures, social links, and other relevant details.

The main goal of SCM is to simplify contact management while providing a smooth and user-friendly experience.

Key Features

  • Secure login system
  • Cloud image upload (Cloudinary)
  • Search and filter contacts
  • Email notifications
  • Modern responsive UI
